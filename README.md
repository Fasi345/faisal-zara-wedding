# Faisal & Zara Wedding Website

A mobile-first static wedding invitation website for GitHub Pages.

## Edit details
Open `index.html` and replace:
- Mehndi, Barat, and Walima timings
- Venue names
- The `href="#"` values for location buttons with Google Maps links
- Chachu names in the **Waiting Eyes** section

## Memorial photo
Add the photo at:

`assets/ustad-zaheer.jpg`

The photo will automatically appear in the memorial section.

## Optional music
Add an MP3 file at:

`assets/music.mp3`

## Publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload `index.html`, `README.md`, and the `assets` folder.
3. Open **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Choose `main` and `/root`, then save.
